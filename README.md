Analisis del comportamiento del cliente

Proyecto: Análisis del Comportamiento del Consumidor Proyecto transversal desarrollado con Python, SQL, Power BI y Gamma App. A través del procesamiento de datos transaccionales de 3,900 compras en diversas categorías de productos, se analizó el comportamiento de compra para identificar patrones de gasto, segmentación de clientes, preferencias y dinámicas de suscripción. El objetivo central fue generar insights de valor y analítica predictiva para orientar la toma de decisiones empresariales estratégicas.
